# Final exam
this project is a simulated simple delivery service.

## Features
### The Person class
* **Create Person:** create a people with name.
* **Introduce:** introduce himself/herself by his/her name.
### Costumer clsss
* **Create Costumer:** create a costumer which it a inherence of persoon.
* **Place Order:** order an items and return object from delivery order class.
### Driver class
* **Create Driver:** create a driver which it a inherence of person class.
* **Deliver:** deliver an item to the customer.
### DeliveryOrder class
* **Create Order:** create a order from a customer and item that customer want.
* **Assign Driver:** return a Driver.
* **Summary:** show summary of the order.
